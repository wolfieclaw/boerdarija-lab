# BOERDARIJA Scenarios Plan

This document defines the first scenario-driven layer for BOERDARIJA.

The goal is to move the product from a generic course site toward a **Casablanca Darija survival + social fluency tool**.

## Design principles

Each scenario should teach:
- what people are likely to say in real life
- what you can safely say back
- one more natural version
- one slangy / casual option where appropriate
- a few common follow-up replies
- audio for high-value lines

Each scenario can later support:
- mini roleplay
- phrase cards
- listening check
- “natural vs textbook vs slang” contrast
- short cultural notes

## Suggested pilot scenarios to build first

If starting small, build these first:
1. Taxi basics
2. Café ordering
3. Market bargaining
4. Saying “I don’t understand”
5. WhatsApp / texting style

These give the strongest immediate practical value.

## Future content format for each scenario

Each scenario page should eventually include:
- short intro
- 8–15 core phrases
- natural vs textbook vs slang notes
- mini dialogue
- optional audio
- quick practice challenge
- “watch out for this” cultural note

That would make the product feel alive and distinct.
