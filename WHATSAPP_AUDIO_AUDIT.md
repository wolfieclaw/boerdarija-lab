# WhatsApp / Texting Style Audio Audit

First-pass audit for a future **WhatsApp / Texting Style** scenario page.

## Good existing coverage
The repo already has useful modern-life audio for:
- `WhatsApp`
- `Message`
- `Telefon` / `Tilifun`
- `Daba` / now
- `Gheda` / tomorrow
- `M3ttl` / late
- `Hna` / here
- `Fayn?` / where?
- `Ja` / come
- `Belati, tsenna shwiya` / wait a bit

## What this means
We can build a meaningful first texting-style page now, even if some of the best full-line conversational phrases still need future audio.

## Likely worth better scenario-native audio later
- `Fink?` / Where are you?
- `Ana jay.` / I’m coming.
- `Ana hna.` / I’m here.
- `Sift lia location.` / Send me your location.
- `3ayyet lia.` / Call me.
- `Ana barra.` / I’m outside.
- `Ana m3ttl shwiya.` / I’m a bit late.

## Recommendation
Build the page now around high-value texting situations using existing audio where possible, then add better full-line social/texting audio later if the scenario proves valuable.
