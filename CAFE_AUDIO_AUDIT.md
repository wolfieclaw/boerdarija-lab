# Café Ordering Audio Audit

First-pass audit for a future **Café Ordering** scenario page.

## Good existing coverage
The repo already has useful café-related audio for:
- `Atay` / tea
- `Qhwa` / coffee
- `Qr3a dyal lma` / bottle of water
- `Hlib` / milk
- `Khobz` / bread
- `Fatura` / bill / invoice
- `Skkar` / sugar
- `Bghit` / I want
- `Bghina` / we want
- `Shnu bghiti?` / what do you want?
- `Wash bghiti l-qhwa?` / do you want coffee?
- `Bghit l-ma` / I want water

## What this means
We can build a first Café Ordering scenario page right now without new TTS generation.

## Likely missing / still rough
These would eventually deserve cleaner scenario-native audio:
- `Bghit atay, 3afak.`
- `Bghit qhwa, bla skkar.`
- `Jib lia l-fatura, 3afak.`
- `Wash kayn chi haja nakol?`
- `Zid qlil d skkar.`

## Recommendation
Build the page now using existing audio where available and mark any missing full-line café phrases for later generation.
