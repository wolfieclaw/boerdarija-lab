# Market Bargaining Audio Audit

First-pass audit for a future **Market Bargaining** scenario page.

## Good existing coverage
The repo already has useful market-related audio for:
- `Kam?` / how much?
- `Kam hada?` / how much is this?
- `Ghali` / expensive
- `Rkhis` / cheap
- `Souk` / market
- `Hanout` / shop
- `Flus` / money
- `Bghit` / I want
- `Neqqs shwiya, 3afak` / reduce a little, please
- `Bsh7al hada?` / how much is this? (lesson audio)
- `Ghali bzzaf` / very expensive
- `Mashi ghali` / not expensive

## What this means
We can build a strong first market scenario page without new TTS.

## Still worth better scenario-native audio later
- `Kam hada, 3afak?`
- `Ghali bzzaf.`
- `Neqqs shwiya.`
- `Ma 3ndish بزاف د الفلوس.` / I don’t have much money.
- `Safi, khllih.` / okay, leave it.
- `Bghit hada.` / I want this one.

## Recommendation
Build the page now using existing audio and natural phrase support. Mark missing full-line bargaining phrases for later TTS if needed.
